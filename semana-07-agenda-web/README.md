# Semana 7: Agenda Web con Vaadin

## Descripción
Primera aplicación web con Spring Boot y Vaadin. Muestra dos vistas navegables.

## Cómo ejecutar

mvn spring-boot:run
Luego abrir http://localhost:8080

## Vistas

/: Pagina de inicio con titulo, descripcion y boton a contactos
/contactos: Pagina de contactos con mensaje y boton para volver, con una notificacion que aparece y desaparece
